# Definition

A [[data type]] that allows a program to create [[set]] and to apply the mathematical operations defined in [[set]] theory.

# Notes

For example:
```
TYPE LetterSet = SET OF CHAR
DEFINE Vowels ('A', 'E', 'I', 'O', 'U'): LetterSet
```
